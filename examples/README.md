# Examples

This directory contains small runnable examples for the public repository.

## Included

- `python_sdk/basic_normalize.py` - normalize inline rows with the Python API
- `fhir_ingest/normalize_bundle.py` - ingest a tracked FHIR bundle fixture
- `rest_api/curl_examples.md` - minimal curl calls for the REST server
- `custom_aliases/use_custom_aliases.py` - load a custom alias file and normalize a vendor-specific name

These examples are intentionally small and depend only on tracked files in the repository.
